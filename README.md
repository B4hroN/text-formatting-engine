# Text Formatting Engine

Модульный движок форматирования текста на Modern C++17. Поддерживает переносы слов с учётом пунктуации, выравнивание и настраиваемые границы вывода.

Выделен в отдельный модуль из проекта текстовой RPG для обеспечения чистой архитектуры и переиспользуемости кода.

## Возможности
- **Умные переносы:** Разбиение текста с учётом знаков препинания, пробелов и длины строки.
- **Выравнивание:** Left, Right, Center с корректным распределением отступов.
- **Безопасность:** RAII, управление ресурсами без утечек, `std::string_view` для минимизации копирований.
- **Модульность:** Разделение ответственности (IO, парсинг, вывод), легкая интеграция в консольные приложения.
- **Конфигурация:** Гибкая настройка ширины, декораторов и гарантированных отступов.

## Стек
- **C++17**
- **STL:** `algorithm`, `vector`, `string_view`, `fstream`
- **Паттерны:** ООП, шаблоны, функторы

## Использование
```cpp
#include "print.hpp"

// Вывод текста с выравниванием и символами границ
print("Пример текста для форматирования", TextAlignment::Left, '.', '.');
```

## Сборка
```bash
g++ -std=c++17 -O2 main.cpp print.cpp read.cpp settings.cpp -o engine
./engine
```
